## 🎯 About ProTrack

ProTrack is a modern web-based project management dashboard designed to help teams plan, track, and deliver work efficiently. By combining task tracking, progress monitoring, and real-time analytics, ProTrack enables better collaboration, clearer insights, and improved project execution.

### 🌟 Why ProTrack?

- **Data-Driven Decisions**: Make informed decisions using real-time project analytics and insights
- **Goal-Focused Execution**: Set clear project goals and track progress visually
- **Centralized Tracking**: Monitor tasks, timelines, and team activities in one place
- **Actionable Insights**: Gain clarity on project performance through detailed reports and metrics

## 🚀 Quick Start

### Option 1: Simple Setup (No Dependencies)
```bash
# Clone the repository
git clone https://github.com/abhinavrathee/ProTrack.git

# Navigate to project directory
cd ProTrack

# Open in browser
open index.html
```

### Option 2: Development Setup (With React)
```bash
# Clone and install
git clone https://github.com/abhinavrathee/ProTrack.git
cd ProTrack
npm install

# Start development server
npm start

# Open your browser
# Navigate to: http://localhost:3000
```
